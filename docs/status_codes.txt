ELM Module Status Codes:

"-100": OK
"-101": User Authenication Error (Login Not found)
"-102": Access denied for the content
'-103": Object creation failed due to either missing Leaf or Object already exists.
"-104": Object does not exists
"-105": Combination of -102 and -103
"-106": issue while parsing.
"-111": Error or Exception raised
"-121": Middleware failed.
